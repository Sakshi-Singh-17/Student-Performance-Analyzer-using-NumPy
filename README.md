# 📊 Student Performance Analyzer (NumPy)

## 📌 Overview
This project analyzes student academic performance using NumPy.  
It calculates averages, assigns grades, identifies toppers, and evaluates subject-wise performance.

---

## 🚀 Features
- 📈 Calculate average marks of each student  
- 🏆 Identify the topper based on total marks  
- 🎯 Assign grades (A, B, C) based on performance  
- ❌ Detect pass/fail status with subject-wise failure  
- 📊 Find highest scoring subject  

---

## 🛠️ Technologies Used
- Python  
- NumPy  

---

## 📂 Dataset
The dataset consists of:
- 5 students  
- 4 subjects (English, Maths, Physics, Chemistry)  
- Marks stored in a NumPy array  

---

## 🧠 Concepts Used
- NumPy arrays  
- Indexing and slicing  
- Aggregation functions (`mean`, `sum`, `argmax`)  
- Loops and conditional statements  
- Basic data analysis
